# ElectroExpress API

## Consulatar stock:

`http://127.0.0.1:5000/stock`

Parámetros:

- `brand`: filtrar por marca (`http://127.0.0.1:5000/stock?brand=apple`)
- `price_lowereq`: filtrar por precio menor o igual (`http://127.0.0.1:5000/stock?price_lowereq=200`)
